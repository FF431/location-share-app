# 位置共享APP

一个基于Web技术的位置共享应用，用于实时共享两个手机的位置，并通过AI分析位置情况。

## 功能特点

- **实时定位**：使用浏览器的地理位置API获取高精度位置
- **位置共享**：通过服务器实时同步两个用户的位置
- **地图显示**：使用Leaflet.js显示地图和标记
- **AI分析**：分析两个位置的距离、可能的活动和提供智能建议
- **响应式设计**：适配不同屏幕尺寸的移动设备

## 技术栈

- **前端**：HTML5, CSS3, JavaScript, Leaflet.js
- **后端**：Python, HTTP Server
- **地图**：OpenStreetMap
- **地理位置**：浏览器Geolocation API
- **AI分析**：DeepSeek API

## 使用方法

### 1. 启动后端服务器

```bash
# 进入backend目录
cd backend

# 启动服务器
python server.py

# 服务器将在 http://localhost:8000 运行
```

### 2. 打开前端页面

在两个不同的浏览器或设备上打开 `frontend/index.html` 文件。

### 3. 开始共享位置

1. 在每个设备上输入不同的用户ID（例如：用户1输入"user1"，用户2输入"user2"）
2. 在好友ID字段中输入对方的用户ID
3. 点击"开始共享"按钮
4. 允许浏览器访问位置信息
5. 地图上将显示两个用户的实时位置
6. AI分析区域将显示距离分析和智能建议

## 注意事项

- 确保浏览器支持Geolocation API
- 确保设备开启了位置服务
- 为了获得最佳效果，建议在室外或信号良好的环境中使用
- 后端服务器需要保持运行状态以实现位置同步
